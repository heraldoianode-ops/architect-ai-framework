1

Instalación inicial

una sola vez en tu PC · Windows

⚡Este proceso se hace una única vez. Nunca más necesitás repetirlo.
1

Verificar requisitos

Abrí una terminal (CMD o PowerShell) y verificá que tenés instalado:

git --version    # debe responder: git version X.X
claude --version # debe responder: claude vX.X
HerramientaSi no está instalada
githttps://git-scm.com/download/win
claudehttps://claude.ai/code
2

Descargar y ejecutar el setup

Descargá setup-windows.bat del repositorio:

https://github.com/heraldoianode-ops/architect-ai-framework
▶Doble click en setup-windows.bat — no necesita configuración previa.

Cuando pregunte por la ruta confirmás con S + Enter. El script hace todo solo.

3

Resultado en tu PC

C:\heraldoianode-ops\ ├── architect-ai-framework\ ← plantilla base (no tocar) ├── architect-ai-modules\ ← library de módulos compartido ├── projects\ ← AQUÍ van todos tus proyectos └── .architect-ai-paths.json ← registro de rutas
2

Iniciar un proyecto nuevo

una vez por cada programa · 5–10 minutos

1

Crear el repositorio en GitHub

Ir a github.com/new y crear el repo:

Configuración del repo
Nombrenombre-de-tu-proyecto
VisibilidadPrivate o Public
READMENO inicializar — dejarlo vacío
2

Clonar el framework como base

cd C:\heraldoianode-ops\projects
git clone C:\heraldoianode-ops\architect-ai-framework nombre-de-tu-proyecto
cd nombre-de-tu-proyecto
3

Conectar al repo de GitHub

git remote remove origin
git remote add origin https://github.com/heraldoianode-ops/nombre-de-tu-proyecto.git
git push -u origin main
4

Instalar git hooks

copy .git-hooks\pre-commit  .git\hooks\pre-commit
copy .git-hooks\post-commit .git\hooks\post-commit
5

Abrir Claude Code e iniciar el proyecto

claude

Claude Code lee CLAUDE.md automáticamente. Luego escribís:

vos
INICIAR PROYECTO nombre-de-tu-proyecto
AI
Tipo de proyecto? 1) SaaS Web 2) API REST 3) Mobile 4) Windows 5) Next.js 6) NPM 7) Custom
vos
1
AI
nombre-de-tu-proyecto v0.1.0 — 0% Nodo: 1.1 — Setup repo. Arrancamos.
